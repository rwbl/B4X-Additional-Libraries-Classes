# B4X-Additional-Libraries-Classes
Referencies to forum publications of additional libraries and classes developed for B4X by [Anywhere Software](https://www.b4x.com).

## B4X CustomViews
* [xLevelIndicator](https://www.b4x.com/android/forum/threads/b4x-customview-class-xlevelindicator.135894/) [Class] - Monitor the level of a device, i.e. Battery (%), Voltage (V), Tank Level (m3) and more.
